# vimrc-ctrl


## Usage

### install

``` sh
vimrc-ctrl install demo https://github.com/samwhelp/tool-svim.git
```

### list

run

``` sh
vimrc-ctrl list
```

show

```
demo
```

### set

``` sh
vimrc-ctrl set demo
```


### get

run

``` sh
vimrc-ctrl get
```

show

```
demo
```

### remove

``` sh
vimrc-ctrl remove demo
```
