# xfted


## 專案連結

## 使用說明

### 搭配檔案管理器

1. 開啟檔案管理器，例如使用 `thunar`
2. 點選某個檔案
3. 滑鼠右鍵，選擇開啟使用「Xfted」，
4. 就會開啟 xfce4-terminal，並且開啟新的分頁，並且使用「EDITOR」所設定的「文字編輯器」開啟該檔案

### 在xfce4-terminal下指令

1. 開啟 `xfce4-terminal`
2. 執行 `xfted file`, 就會開啟新的分頁，並且使用「EDITOR」所設定的「文字編輯器」開啟該檔案
