
# Xftt

## Install

``` sh
make install
```

* [xftt](xftt)
* [xftt.desktop](xftt.desktop)

## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific File Or Dir
3. Open With Xftt.


### On xfce4-terminal

1. Launch `xfce4-terminal`
2. Run `xftt file` or `xftt dir`
