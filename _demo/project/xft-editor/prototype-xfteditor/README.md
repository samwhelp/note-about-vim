## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific File
3. Open With Xfted

### On xfce4-terminal

1. Launch `xfce4-terminal`
2. Run `xfted file`
