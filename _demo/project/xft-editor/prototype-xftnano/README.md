
# Xftnano


## Target Editor

* [nano-editor](https://www.nano-editor.org/)


## Install

``` sh
make install
```

* [xftnano](xftnano)
* [xftnano.desktop](xftnano.desktop)

## Remove

``` sh
make remove
```

## Usage

### On File Manager

1. Launch FileManager. For example: `thunar`
2. Specific File
3. Open With Xftnano

### On xfce4-terminal

1. Launch `xfce4-terminal`
2. Run `xftnano file`
